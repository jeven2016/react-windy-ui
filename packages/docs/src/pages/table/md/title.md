## 表格 Table [editUrl]

Table 常用来展示多行多列的数据，在企业应用中广泛被使用。

Table 提供的功能主要有：

- 提供多种类型的 Table 可供切换
- 支持表格的排序、过滤、单选、多选等功能
- 支持固定表头、固定列的滚动展示
- 基于 JSON 数据生成完整的表格
