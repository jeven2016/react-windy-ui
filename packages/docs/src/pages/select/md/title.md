## 选择 Select [editUrl]

Select 与 Dropdown 组件有相似之处，但 Select 会显示上次选中的选项而 Dropdown 只能单击且不会显示选中状态，

Select 主要有以下功能

- 单选 Select
- 多选 Select
- 支持关键字搜索
- Select 的菜单也可以多个方位显示
