## 选择 Select [editUrl]    
Select与Dropdown组件有相似之处，但Select会显示上次选中的选项而Dropdown只能单击且不会显示选中状态，

Select主要有以下功能

- 单选Select
- 多选Select
- 支持关键字搜索
- Select的菜单也可以多个方位显示
