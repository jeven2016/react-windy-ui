+++ zh_CN   
按钮中可以通过设置<Code>loading</Code>属性，展示等待中状态。默认情况下<Code>Loader</Code>在左侧显示，
如果需要在右侧显示则可以将<Code>leftLoader</Code>属性设置为<Code>false</Code>。 如果需要自定义组件，
可以通过<Code>loader</Code>属性配置自己所需要的<Code>loader</Code>。
