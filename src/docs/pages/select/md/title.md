## 选择 Select [editUrl]    
Select与Dropdown组件有相似之处，不同之处就在于，Dropdown只能单击某个选项而且不会高亮显示，
但Select却可以。

Select主要有以下功能
- 单选Select
- 多选Select
- 支持关键字搜索
- Select的菜单也可以在十二个方位显示
